# Software-Architecture-Clustering
Comparative analysis of clustering algorithms and LLMs for software architecture recovery.
The final output rsf files are in output/*.rsf.

To run the sample.sh file, it is necessary to use chmod +x sample.sh to provide execute permissions.
To install the dependencies in dsse_week2_group11_llm, use the command below with Python 3.10.12 already installed in the system
        pip install -r requirements.txt

It is necessary to create a separate virtual environment named "venv" inside the dsse_week2_group11_llm directory for the code to run as it is in the HPC environment. If not, respective changes to the configuration within the file in the form of variable assignments are required.
It is also required to put the HF_TOKEN in the sample.sh.
